# Claude Code 新手友善包（Claude for Everyone）

> 給**完全沒有寫程式基礎**的人用的 Claude Code 設定。
> 讓 AI 助理講人話、會提醒你、不會亂動你的重要檔案，畫面下方也換成看得懂的白話。

---

## 這是什麼？（一句話）

Claude Code 本來是設計給工程師的，預設畫面充滿術語。
這個資料夾把它「重新裝潢」成新手模式——**像把一台專業相機調成傻瓜模式**，
你不用懂任何技術，也能安心使用。

## 它幫你做了什麼？

| 小幫手 | 白話說明 |
| --- | --- |
| 🗣️ **講人話** | 每次動手前先用一句白話說「我正要做什麼」，全程白話＋比喻，不丟一堆程式碼給你 |
| 🔔 **語音提醒** | 工作做完、或需要你決定時，電腦會出聲叫你回來看，不用一直盯著螢幕 |
| 🔒 **資料夾安全鎖** | 你指定的重要資料夾，AI 只能讀、不能改不能刪，避免誤動 |
| 🧠 **記憶** | 把「你是誰、你的習慣」長期記住，不用每次重講 |
| 📊 **白話狀態列** | 畫面最下方那行，從術語換成人話：現在用哪個助理、在處理哪個資料夾、對話空間還剩多少 |

## 怎麼用？（最簡單的方法）

你**不需要自己動手裝**。把這個資料夾下載到電腦後，打開 Claude Code，
直接對它說這句話就好：

> **「請照這個資料夾裡的說明，把我的 Claude Code 設定成新手友善模式。」**

Claude 會一步步帶你完成，過程中需要你決定的（例如要不要保護某個資料夾），
它會用白話問你。

想自己一步步來、或想知道每個設定在做什麼，看 👉 [docs/01-開始使用.md](docs/01-開始使用.md)

## 這個資料夾裡有什麼？

```
settings.example.json      ← 主設定（白話提醒、語音、安全鎖、狀態列都在這）
hooks/folder-guard.sh      ← 資料夾安全鎖
scripts/statusline-plain.sh← 白話狀態列
scripts/notify-telegram.sh ← （選用）把檔案傳到你的 Telegram
memory/                    ← 記憶範本（你是誰、你的習慣）
docs/                      ← 全程白話的說明書
```
