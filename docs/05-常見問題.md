# 05・常見問題

## Q：畫面上那些技術框框（程式碼、Wrote 50 lines…）能完全關掉、不要看到嗎？

**老實說：目前不行。** Claude Code 現在「沒有」把進度框框整個關掉的設定。

你能做的：

- **按 `Ctrl + O`**：切換顯示模式，把一部分框框收合，畫面比較乾淨。
  （但檔案類的操作還是會露出來。）
- **把它當背景雜訊忽略**：這個新手包已經設定成「我每次動手前先用一句白話告訴你
  我正要做什麼」。你只要看那句白話就好——就像坐計程車不用懂引擎，
  聽司機說「快到了」就行。

很多使用者都在敲碗一個「完全隱藏」的開關，但官方還沒開放。等它開放了，
這份說明會更新。

---

## Q：語音提醒太大聲 / 不想要怎麼辦？

直接跟 Claude 說「把語音提醒關掉」或「音量調小一點」，它會幫你改。
（音量預設是 40%。）

---

## Q：我用 Windows / Linux，語音提醒會動嗎？

不會，預設那段是 Mac 專用。跟 Claude 說你的系統，請它換成對應的提醒方式即可。

---

## Q：狀態列那行的數字代表什麼？

- 🧠 **助理大腦**：現在用哪個 AI（Opus 最聰明、Sonnet 又快又好、Haiku 最快）
- 📁 **正在處理**：你現在所在的資料夾名稱
- 💭 **對話空間**：這次對話的「記憶」用了多少（越用越多）。
- 🔄 **距離自動整理**：還剩多少空間才會觸發「自動打包濃縮」。**這個數字會越用越少**，
  逼近 0% 時 Claude 會自動把前面對話濃縮一次、騰出空間。
- ⏳ **這5小時用量**：短期額度用了多少（即時數字，只有 Pro / Max 會顯示）。
- 📅 **這7天用量**：每週額度用了多少（即時數字，只有 Pro / Max 會顯示）。

---

## Q：安全鎖會不會擋到我自己想做的事？

不會擋讀取，只擋「修改 / 刪除 / 搬移」你指定保護的那個資料夾。
而且預設是「沒有保護任何資料夾」——要等你自己指定，它才會啟動。

---

## Q：怎麼開關某個功能？（語音、安全鎖、出廠檢查…）

打 `/設定面板`，它會用白話列出每個功能現在是開還是關，你再用講的告訴它要開 / 關哪一個，
例如「把語音關掉」「幫我保護某個資料夾」。打 `/說明` 則會列出全部功能與指令。

---

## Q：我把設定弄亂了，想回到原本的樣子？

跟 Claude 說「把新手包的設定移除、還原成原本的」，或自己執行 `bash uninstall.sh` 即可。
安裝時就已經自動備份你原本的設定，所以能**完整退回**到安裝前的樣子。
