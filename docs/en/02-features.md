# 02 · How the Six Features Work

> 🌐 [中文](../02-六大功能.md) · **English**

No tech knowledge needed — here's what each feature does, explained with everyday analogies.

## 1. 🗣️ Plain language
**Like having** a translator on staff who turns engineer jargon into words you understand.

Every time you send a message, the system automatically reminds the AI: "This person is a beginner — say one plain sentence before acting, speak in everyday language, run heavy work in the background, report only results." This reminder fires **every single time**, so it doesn't rely on the AI "remembering" — it's reliable.

## 2. 🔔 Voice reminders
**Like** setting a kitchen timer that calls you when it's ready.

When the AI finishes a task or needs your decision, the computer speaks up to call you back. Default volume is 40%; if it's too loud, just ask it to lower it or turn it off.

## 3. 🔒 Folder safety lock
**Like** locking your important cabinet but leaving one drawer open to put things in.

You can mark a folder as "look but don't touch." It's **off by default** (nothing is locked) and only activates once you specify a folder. See [04 · Folder Safety Lock](04-folder-lock.md).

## 4. 📊 Plain-language status bar
**Like** a car dashboard, but instead of "RPM" and "oil pressure" it says "how many miles you can still drive."

The bottom line of the screen — once full of jargon — is now plain language: which assistant is active, which folder you're in, how much conversation space is used, and live usage numbers. See [03 · The Plain Status Bar](03-status-bar.md).

## 5. 🌱 Grows with you (the soul of this kit)
**Like** an apprentice who understands you more over time and offers, "let me make this automatic for you."

As the AI works with you, it watches your habits — what you **keep correcting**, the preferences you **always ask for**, the workflows you **redo by hand**, even **periodic tasks** — then **proactively asks** whether to turn it into automation (an auto-running rule, a command, a schedule, or a flow). **It only acts if you say yes**, and stops nagging if you say no. You can also run `/習慣盤點` anytime to trigger a review.

> Claude Code can already "remember things" natively (memory). What makes this kit special is upgrading "remember" into "**proactively turn it into automation**," with ready-made plain-language templates so you don't start from a blank page.

## 6. 🔍 Factory check
**Like** a final quality inspection before a product ships — it even checks whether the parts snap together.

Whenever the AI changes any code or setting, it automatically runs a lightweight check: any bugs, any safety risks, plus a "**fit check**" — does the changed part still connect with everything else (looking at the joints, not just one file). It flags issues in plain language and **asks before fixing**. You can also run `/出廠檢查` anytime.

## Meet the four plain-language commands
Type `/` in the Claude Code input box:

- `/說明` — list your features and commands (help)
- `/設定面板` — turn each feature on/off, just by talking
- `/習慣盤點` — find habits worth automating
- `/出廠檢查` — run a lightweight check on recent changes
