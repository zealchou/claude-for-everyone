# 01・開始使用

## 你需要先有什麼？

1. **電腦**：Mac 最好（語音提醒用的是 Mac 內建功能）。Windows / Linux 也能用，
   只是語音提醒那條要小改，可以請 Claude 幫你改。
2. **裝好 Claude Code**：還沒裝的話，去 Claude Code 官網照指示安裝即可。
3. 就這樣，不需要會寫程式。

## 最簡單的裝法（推薦）

把這個資料夾下載到電腦，打開 Claude Code，對它說：

> **「請照這個資料夾裡的說明，把我的 Claude Code 設定成新手友善模式。」**

Claude 會幫你把設定檔放到對的位置、問你要不要保護某個資料夾、再確認一切正常。
**你只要回答它的白話問題就好。**

## 想知道它會把東西放到哪？

Claude Code 的設定都集中在你電腦的一個叫 `.claude` 的資料夾裡（通常在你的「家目錄」）。
這個包會放進去的東西：

- 主設定檔（讓助理講人話、開語音、掛上安全鎖和狀態列）
- 一支「資料夾安全鎖」小程式
- 一支「白話狀態列」小程式
- 記憶範本（你是誰、你的習慣）

## 裝好之後

- **重開一次 Claude Code**，新設定才會生效。
- 畫面最下方那行會變成白話（例如：🧠 助理大腦：Opus（最聰明） · 📁 正在處理：… · 💭 對話空間：用了 12%（充裕））。
- 之後就照平常用。需要你決定的時候，它會出聲提醒並用白話問你。

## 想客製？

- 想保護某個資料夾 → 看 [04・資料夾安全鎖](04-資料夾安全鎖.md)
- 想換狀態列顯示的內容 → 看 [03・白話狀態列](03-白話狀態列.md)
