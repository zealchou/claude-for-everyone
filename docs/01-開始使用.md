# 01・開始使用

> 🌐 **中文** · [English](en/01-getting-started.md)

## 你需要先有什麼？

1. **電腦**：建議用 **Mac**（語音提醒用的是 Mac 內建功能）。Windows 要先裝 WSL 或
   Git Bash 才能跑，比較麻煩，**新手建議優先用 Mac**；語音那條也請 Claude 幫你換成 Windows 版。
2. **裝好 Claude Code**：還沒裝的話，去 Claude Code 官網照指示安裝即可。
3. **兩個小工具 `jq`、`python3`**：多數 Mac 已經內建。萬一缺，直接跟 Claude 說
   「幫我裝 jq」它會處理。（狀態列需要 jq 才會顯示；若你原本已有設定檔、又缺 jq，
   一鍵安裝會自動暫停、不會亂動你的設定。）
4. 就這樣，不需要會寫程式。

## 第一步：先把這個包下載下來

到這個 GitHub 頁面，**右上角點綠色的「Code」按鈕 → 選「Download ZIP」**，下載後
**解壓縮**，你就會得到這個資料夾。（會用 `git` 的人也可以直接 clone。）

## 最簡單的裝法（推薦）

**重點：要在「剛剛解壓出來的那個資料夾裡」打開 Claude Code**（這樣它才看得到說明檔），
然後對它說：

> **「請照這個資料夾裡的說明，把我的 Claude Code 設定成新手友善模式。」**

Claude 會幫你把設定檔放到對的位置、問你要不要保護某個資料夾、再確認一切正常。
**你只要回答它的白話問題就好。**

> 想自己動手的人：在這個資料夾執行 `bash install.sh` 就會一鍵安裝。
> **安裝前會自動備份你原本的設定**，不喜歡隨時用 `bash uninstall.sh` 一鍵還原。

## 想知道它會把東西放到哪？

Claude Code 的設定都集中在你電腦的一個叫 `.claude` 的資料夾裡（通常在你的「家目錄」）。
這個包會放進去的東西：

- 主設定檔（讓助理講人話、開語音、掛上安全鎖和狀態列）
- 一支「資料夾安全鎖」小程式、一支「白話狀態列」小程式
- 五個白話指令（`/說明`、`/設定面板`、`/習慣盤點`、`/出廠檢查`、`/解釋給我聽`）
- 白話記憶與「會自己長大 / 出廠檢查」的長期指令

## 裝好之後

- **重開一次 Claude Code**，新設定才會生效。
- 之後就照平常用。需要你決定的時候，它會出聲提醒並用白話問你。

## 怎麼確認裝成功了？

重開後，用這三個簡單檢查（任一個有反應就代表成功）：

1. **看畫面最下方那三行**：有沒有出現 🧠 📁 💭 這些圖示？有 → 白話狀態列成功。
2. **打 `/說明`**：有沒有跳出功能與指令清單？有 → 指令成功。
3. **隨便請它做件小事**，做完有沒有聽到電腦講話？有 → 語音提醒成功。

> 如果都沒反應，直接跟 Claude 說「**狀態列沒生效，幫我檢查一下**」，它會幫你看。
