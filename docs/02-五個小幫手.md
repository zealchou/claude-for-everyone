# 02・五個小幫手怎麼運作

不用懂技術，這裡用生活化的方式說明每個功能在做什麼。

## 1. 🗣️ 講人話

**像什麼**：請了一個會「翻譯」的助理，把工程師術語轉成你聽得懂的話。

每次你送出一句話，系統就會自動提醒 AI：「對方是新手，動手前先講一句白話、
全程說人話、不要丟程式碼。」這個提醒是**自動的、每次都會發生**，
不靠 AI「臨時記得」，所以很穩定。

## 2. 🔔 語音提醒

**像什麼**：煮東西時設了個鬧鐘，好了會叫你。

當 AI 把工作做完、或需要你做決定時，電腦會用語音說一句話叫你回來看，
你就不用一直盯著螢幕。預設音量 40%，覺得吵可以叫 Claude 調小或關掉。

## 3. 🔒 資料夾安全鎖

**像什麼**：把重要櫃子上鎖，只留一個抽屜可以放東西。

你可以指定某個資料夾「只能看、不能動」。AI 想讀裡面的資料沒問題，
但只要它想修改 / 刪除 / 搬移，就會被當場攔下來。
預設是「沒鎖任何資料夾」，等你指定才生效。設定方式看 [04](04-資料夾安全鎖.md)。

## 4. 🧠 記憶

**像什麼**：一個記得你喜好的老朋友，不用每次自我介紹。

把「你是誰、你的工作、你的習慣」記下來，之後每次對話 AI 都知道。
你不用自己寫——直接跟 Claude 說「記住我是新手、請一直用白話」，
它會幫你建立記憶。範本在 `memory/` 資料夾。

## 5. 📊 白話狀態列

**像什麼**：汽車儀表板，但把「轉速、油壓」換成「還能開幾公里」這種看得懂的話。

畫面最下方那行，原本是一堆術語，現在換成白話：
現在用哪個助理、在處理哪個資料夾、對話空間還剩多少。詳見 [03](03-白話狀態列.md)。
