# 02・六大功能怎麼運作

> 🌐 **中文** · [English](en/02-features.md)

不用懂技術，這裡用生活化的方式說明每個功能在做什麼。

## 1. 🗣️ 講人話
**像什麼**：請了一個會「翻譯」的助理，把工程師術語轉成你聽得懂的話。

每次你送出一句話，系統就會自動提醒 AI：「對方是新手，動手前先講一句白話、全程說人話、複雜的丟背景、只報結果。」這個提醒是**每次都會發生**，不靠 AI「臨時記得」，很穩定。

## 2. 🔔 語音提醒
**像什麼**：煮東西設了鬧鐘，好了會叫你。

當 AI 把工作做完、或需要你做決定時，電腦會用語音叫你回來看。預設音量 40%，覺得吵可以叫它調小或關掉。

## 3. 🔒 資料夾安全鎖
**像什麼**：把重要櫃子上鎖，只留一個抽屜可以放東西。

你可以指定某個資料夾「只能看、不能動」。預設「沒鎖任何資料夾」，等你指定才生效。設定方式看 [04・資料夾安全鎖](04-資料夾安全鎖.md)。

## 4. 📊 白話狀態列
**像什麼**：汽車儀表板，但把「轉速、油壓」換成「還能開幾公里」這種看得懂的話。

畫面最下方那行，原本是術語，現在換成白話：現在用哪個助理、在處理哪個資料夾、對話空間用了多少、用量即時數字。詳見 [03・白話狀態列](03-白話狀態列.md)。

## 5. 🌱 會自己長大（這個包的靈魂）
**像什麼**：一個越用越懂你的學徒，會主動說「這個我來幫你弄成自動的吧」。

AI 一邊跟你互動一邊留意你的習慣——你**重複糾正**的地方、**每次都要**的偏好、**一直手動重做**的流程——然後**主動問你**要不要把它變成自動機關（一個會自動執行的規則、一個指令、或一段流程）。**你說好它才做**，說不用就不再囉嗦。你也可以隨時打 `/習慣盤點` 叫它幫你盤點一次。

> 這部分本來就是 Claude Code 原生就能「記住事情」（記憶功能）。這個包的特別之處，是**把「記住」升級成「主動幫你變成自動」**，而且附了白話範本，省得你對著空白檔不知從何寫起。

## 6. 🔍 出廠檢查
**像什麼**：產品出貨前的最後品管，連「零件接得上嗎」都檢查。

只要 AI 改了任何程式 / 設定，做完會自動做一次輕量檢查：會不會出錯、安不安全，還有「**卡榫檢查**」——剛改的東西跟原本的其他部分**還接不接得上**（不只看單一檔案，看接合處）。有問題先用白話跟你說，**要不要修先問你**。你也可以隨時打 `/出廠檢查`。

---

## 順便認識四個白話指令
在 Claude Code 輸入框打 `/` 就能用：

- `/說明` — 列出你有哪些功能、哪些指令（help）
- `/設定面板` — 逐條開關每個功能，用講的就能操作
- `/習慣盤點` — 找出可以幫你自動化的習慣
- `/出廠檢查` — 對剛改的東西做一次輕量檢查
