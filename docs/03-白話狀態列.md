# 03・白話狀態列

> 🌐 **中文** · [English](en/03-status-bar.md)

「狀態列」就是 Claude Code 畫面**最下方那幾行**。預設的版本充滿術語
（model、context %、token…），新手根本看不懂。這個包把它換成白話，
而且**分成三行**——擠成一長行時，後面的字會被畫面切掉看不到，分三行就一目了然。

## 換完之後長這樣（三行）

```
🧠 助理大腦：Opus（最聰明）   ·   📁 正在處理：我的專案
💭 對話空間：用了 12%（充裕）   ·   🔄 距離自動整理：還剩 88%（很充裕）
⏳ 這5小時用量：41%（還很夠）   ·   📅 這7天用量：68%（用得差不多了）
```

三行各管一件事，掃一眼就懂：

- **第 1 行・身份**：現在用哪個助理大腦、正在處理哪個資料夾。
- **第 2 行・對話**：這次對話的記憶空間用了多少、距離自動整理還剩多少。
- **第 3 行・額度**：這 5 小時 / 7 天的額度用了多少（只有 Pro / Max 才有數字；
  免費方案這一行會顯示一句溫和提示，畫面一樣維持三行不亂掉）。

每一段的意思：

| 顯示 | 白話意思 |
| --- | --- |
| 🧠 助理大腦 | 現在用哪個 AI。Opus＝最聰明、Sonnet＝又快又好、Haiku＝反應最快、Fable＝最新款 |
| 📁 正在處理 | 你現在所在的資料夾名稱 |
| 💭 對話空間 | 這次對話的「記憶」用了多少（越用越多） |
| 🔄 距離自動整理 | 還剩多少空間才會觸發「自動打包濃縮」。**這個數字會越用越少**，像油表。逼近 0% 時，Claude 會自動把前面對話濃縮一次騰出空間 |
| ⏳ 這5小時用量 | 短期額度用了多少（**即時數字**，Pro / Max 才會顯示） |
| 📅 這7天用量 | 每週額度用了多少（即時數字，Pro / Max 才會顯示） |

> 沒有訂閱方案的人，後面兩段（用量）會自動不顯示，不會出錯。

## 用量數字多久更新一次？

設定裡有一行讓它**每 10 秒自動刷新**，所以用量是接近即時的。
（這些用量數字要等你跟 AI 對話、有第一次回應之後才會出現。）

## 想改顯示內容？

直接跟 Claude 說，例如：
- 「狀態列幫我加上今天花了多少錢」
- 「我不想看 7 天用量，拿掉」
- 「字太長了，幫我精簡」

直接跟它說就好，它知道要改裝好後生效的那份（在 `~/.claude/statusline-plain.sh`）。改完**重開 Claude Code** 就生效。
