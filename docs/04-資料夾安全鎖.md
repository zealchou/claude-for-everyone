# 04・資料夾安全鎖

## 它在做什麼？

把你**最重要、絕對不想被誤動**的資料夾保護起來：
AI 可以「讀」裡面的東西，但只要它想「修改 / 刪除 / 搬移 / 覆蓋」，就會被當場攔下來。

**像什麼**：把重要櫃子上鎖，AI 拿鑰匙能打開來看，但不能動裡面的東西；
你可以另外留一個「暫存抽屜」讓它放東西。

## 預設是「沒鎖任何東西」

裝好之後，安全鎖預設**不保護任何資料夾**（很安全，不會擋到你）。
要等你指定一個資料夾，它才會啟動。

## 怎麼指定要保護的資料夾？

最簡單：直接跟 Claude 說，例如：

> 「幫我把『我的硬碟裡的公司資料夾』設成唯讀保護，只留『暫存』那個夾可以寫。」

你只要講出「**資料夾名字**」就好，Claude 知道要去改裝好後生效的那份安全鎖
（在 `~/.claude/hooks/folder-guard.sh`），裡面有兩個設定：

- **保護的資料夾**：整個鎖起來只能讀（要用完整路徑，不確定就把資料夾拖進來給 Claude，它會處理）
- **唯一可寫入的子資料夾**：留一個抽屜給它放東西（不需要的話可以不留）

## 它怎麼判斷要不要攔？

- 你只是**讀取 / 查看** → 永遠放行，不打擾你
- AI 想**改、刪、搬、覆蓋**保護區裡的東西 → 攔下來，並用白話說明原因
- 動到「可寫入子資料夾」 → 放行

設計上是「寧可多攔，不可放過」——保護你的資料優先。

## 想解除或調整？

跟 Claude 說「把安全鎖拿掉」或「改成保護另一個資料夾」即可。
