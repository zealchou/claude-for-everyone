description: 盤點我最近的使用習慣，提議可以自動化的地方（hook / skill / 工作流），我同意才設定
---

請幫我做一次「習慣盤點」，全程白話、不要貼技術細節：

1. 回顧我們最近的互動、以及我的記憶（特別是 feedback 類），找出：
   - 我**重複糾正**你的地方
   - 我**每次都要求**的固定偏好 / 做法
   - 我**一直手動重做**、或老是踩到的同一個雷

2. 把找到的「可以自動化的機會」用白話列成清單，每一項說明：
   - 我常做 / 常要求的是什麼
   - 可以變成哪種自動機關：
     - **hook**＝每次都自動執行的規則（例如「每次回報都用白話」）
     - **skill / 指令**＝我喊一聲就能用的技能
     - **工作流**＝一連串自動完成的多步驟流程
   - 變成自動之後對我有什麼好處、有沒有風險

3. **一項一項問我要不要做**。
   - 我說「好」的，你才動手設定（用 update-config 設 hook 等），設好後用白話告訴我「以後會自動怎樣」。
   - 我說「不用」的就跳過，記在記憶裡別再重提。
   - **任何有風險的設定，動手前一定先跟我確認。**
